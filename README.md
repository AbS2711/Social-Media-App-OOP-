# Social-Media-App-OOP-
This is an app that replicates the structure of a communication application. It displays hardcoded data but has all the connections established between different features. Strictly built using all the OOP principles and basics


Social Media App (OOP Implementation) 📱
This project is a C++ console-based social media application developed as part of a Semester 2 coursework to demonstrate the practical application of core Object-Oriented Programming (OOP) principles. The application provides a modular and scalable design to simulate essential social networking functionalities.

🚀 Key Features
User Profile Management: Create and manage distinct user profiles.

Post Creation: Functionality for users to create and share posts within the network.

Friend Connections: Establish and manage connections between different user profiles.

Modular Architecture: Designed with maintainability and extensibility in mind using OOP concepts.

🧠 OOP Concepts Applied
The application is built upon four fundamental pillars of Object-Oriented Programming:

Encapsulation: Data and methods are bundled into classes (e.g., User, Post) to protect internal states and ensure data integrity.

Inheritance: Used to create specialized objects from base classes, promoting code reusability.

Polymorphism: Implemented to allow different classes to be treated as instances of the same base class through method overriding and interfaces.

Classes & Objects: Every major entity in the app is represented as a class, creating a clear and structured blueprint for system components.

🛠️ Technical Stack
Language: C++

Paradigm: Object-Oriented Programming (OOP)

Development Environment: Visual Studio / VS Code

📂 Repository Structure
*.cpp & *.h: Source and header files containing class definitions and application logic.

.gitignore: Optimized to exclude compiled binaries (.exe, .o, .obj), IDE-specific metadata (.vscode, .vs), and temporary build files to keep the repository clean.
